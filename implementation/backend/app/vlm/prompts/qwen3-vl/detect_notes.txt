Identify the bounding boxes of any general-notes regions — paragraphs of written instructions. Return JSON only.

Coordinates are normalized 0-1 of the page; (0,0) is top-left.

Schema:
{"bboxes": [[x0, y0, x1, y1], ...]}

Notes are prose text blocks (e.g. "GENERAL NOTES", numbered instructions). Do NOT include symbol-key tables, equipment schedules, or the title block. If multiple notes blocks form one column, return ONE bbox covering the column. Empty list if no notes.
