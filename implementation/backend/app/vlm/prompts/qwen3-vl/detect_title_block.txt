Identify the bounding box of the drawing's title bar, banner, or sheet-metadata box. Return JSON only.

Coordinates are normalized 0-1 of the page; (0,0) is top-left.

Schema:
{"bbox": [x0, y0, x1, y1]}   or   {"bbox": null}

Include the title banner AND any project / drawn-by / date / scale metadata box together in ONE bbox. Do NOT return whole-page bboxes. If the drawing has no title block, return null.
