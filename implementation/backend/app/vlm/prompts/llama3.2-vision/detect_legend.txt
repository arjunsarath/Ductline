Identify the bounding boxes of any symbol-key or abbreviation-table regions. Return JSON only.

Coordinates are normalized 0-1 of the page; (0,0) is top-left.

Schema:
{"bboxes": [[x0, y0, x1, y1], ...]}

If the legend is split into multiple separated blocks (e.g. one with symbols, one with abbreviations), return ONE bbox per block. If the drawing has no legend, return an empty list.
